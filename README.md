textcord is a communicator created with the idea of fast text communication within a local network; it is distinguished by speed, lightness, reliability, and ease of use.

It has:
System messages
Real-time message updates
Message reporting system
Ability to block users, permanently, temporarily, and management of this
Login, report, ban logs, etc.

NOTE:
The project is in the development phase; the installer and service were tested on DEBIAN,
currently, the installer does not correctly install the DNS domain and SSL certificate; for these things to work, interference with the DNS service and certificates, as well as among others NGINX, is required.
