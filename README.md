textcord is a communicator created with the idea of fast text communication within a local network; it is characterized by speed, lightness, reliability, and ease of use.

It has:
System messages
Real-time message updates
Message reporting system
Ability to block users, permanently, temporarily, and manage this
Logs of logins, reports, bans, etc.

Updates:
-Fixed own message options
-Fixed account recovery from file
-Installation with DNS and Certificate is no longer problematic, the generated certificate should go to the user's home directory, and to avoid a certificate error on the client, it must be imported on the client computer
